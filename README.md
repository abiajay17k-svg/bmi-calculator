# BMI Calculator

A simple and user-friendly desktop application to calculate Body Mass Index (BMI) and classify health status using Python.

## Features

- Calculate BMI using height (meters) and weight (kilograms)
- Instant health classification based on BMI value
- Provides personalized health suggestions
- Handles invalid inputs gracefully

## Health Categories

| BMI Range | Category |
|-----------|----------|
| < 18.5 | Underweight |
| 18.5 - 24.9 | Normal weight |
| 25 - 29.9 | Overweight |
| ≥ 30 | Obesity |

## Sample Output
Welcome to the BMI Calculator
Enter your height in meters: 1.65
Enter your weight in kilograms: 60

Your BMI is: 22.04
Health Suggestion: Normal weight – Keep up the good work and maintain a healthy lifestyle.

## Technologies Used

- Python 3
- Standard libraries only (no external dependencies)

## How to Run

1. Make sure Python is installed on your system
2. Download `bmi_calculator.py`
3. Open terminal/command prompt
4. Run: `python bmi_calculator.py`
